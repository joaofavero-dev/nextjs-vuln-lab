# 🔐 Test Project – Vulnerability Assessment Lab

This repository is a test project focused on the identification, analysis, and controlled exploitation of vulnerabilities in applications and simulated environments.

---

## 🎯 Objective

The goal of this project is to:

- Demonstrate common vulnerabilities in web applications  
- Test security tools  
- Practice offensive security analysis techniques (pentesting)  
- Study vulnerabilities based on the OWASP Top 10  

---

## 🛠️ Included Vulnerabilities

This project may contain examples such as:

- SQL Injection  
- Cross-Site Scripting (XSS)  
- Broken Authentication  
- Insecure Direct Object Reference (IDOR)  
- Security Misconfigurations  
- Insecure File Upload  

---

## 🧪 Purpose

⚠️ This project is intended exclusively for educational and controlled testing purposes.

It must not be used against real systems without explicit authorization.

---

## 👨‍💻 Target Audience

- Information Security Students  
- Cybersecurity Professionals  
- Developers interested in security  
- Beginner Pentesters
